# `@channel-io/prettier-config`

Shared Prettier configuration

## Installation

### npm

```sh
$ npm install --dev @channel.io/prettier-config
```

### Yarn

```sh
$ yarn add --dev @channel.io/prettier-config
```

## Usage

Channelsâ€™s shared prettier config comes bundled in `@channel-io/prettier-config`. To enable these rules, add a `prettier` property in your `package.json` and reference this shared config as follows:

```JSON

```
